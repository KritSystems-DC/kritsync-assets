# KritSync Assets

Official brand assets for **KritSync — Automation App**.

KritSync is a premium Discord deployment and automation app designed to create polished, client-ready Discord servers for real communities, businesses, creators, game servers, courses, marketplaces, and memberships.

## Brand Direction

KritSync uses a premium gold and black visual identity with a clean, professional automation-app style.

The main logo features:

- A friendly futuristic robot mascot
- Gold and black metallic styling
- A sync/refresh icon
- A database/server stack
- Dark charcoal background
- Premium 3D automation-app presentation

## GitHub Pages URLs

After GitHub Pages is enabled, use these in your KritSync bot .env file:

KRITSYNC_LOGO_URL=https://YOUR_GITHUB_USERNAME.github.io/kritsync-assets/assets/kritsync-logo.png

KRITSYNC_ICON_URL=https://YOUR_GITHUB_USERNAME.github.io/kritsync-assets/assets/kritsync-icon.png

KRITSYNC_BANNER_URL=https://YOUR_GITHUB_USERNAME.github.io/kritsync-assets/assets/kritsync-banner.png

Replace YOUR_GITHUB_USERNAME with your actual GitHub username.
